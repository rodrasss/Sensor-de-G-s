# Sensor-de-G-s
Criamos uma aplicação com ESP32 e sensores de gás no simulador Wokwi para monitorar a qualidade do ar em ambientes internos. O ESP32 coleta dados, os envia para um servidor IoT e exibe os níveis de poluentes em um dashboard em tempo real, ativando dispositivos para melhorar a qualidade do ar automaticamente.
